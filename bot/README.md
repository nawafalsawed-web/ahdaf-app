# بوت أهداف — إنشاء قروبات واتساب تلقائياً

بوت صغير يربط برقم واتساب، ويستقبل طلبات من تطبيق أهداف لإنشاء قروب تلقائياً.

## ⚠️ تنبيه مهم
يستخدم مكتبة **whatsapp-web.js** (غير رسمية من واتساب). إنشاء قروبات كثيرة بسرعة
قد يؤدي إلى **حظر الرقم**. استخدمه بحذر، ويُفضّل رقم ثانوي.

## التشغيل (مرة وحدة)

1. تأكد أن Node.js مثبّت:
   ```
   node -v
   ```
   إذا ما طلع رقم إصدار، ثبّت Node من https://nodejs.org

2. ثبّت المكتبات (داخل مجلد bot):
   ```
   cd ~/Desktop/ahdaf-app/bot
   npm install
   ```
   (أول مرة تأخذ دقايق — تنزّل متصفح Chromium.)

3. شغّل البوت:
   ```
   npm start
   ```

4. راح يطلع **QR code** في الترمنال.
   افتح واتساب في جوالك → الإعدادات → الأجهزة المرتبطة → ربط جهاز → امسح الكود.

5. لما تشوف «🤖 البوت جاهز» — خلاص، البوت شغّال.
   لا تسكّر نافذة الترمنال (البوت لازم يضل شغّال).

## بعدها
- افتح تطبيق أهداف → الإعدادات (⚙️) → فحص اتصال البوت → لازم يطلع «جاهز».
- حط أرقام فريق أهداف في الإعدادات (تنضاف لكل قروب).
- بإضافة عميل، اضغط «🤖 أنشئ القروب تلقائياً» — والباقي على البوت.

## ملاحظات
- الجلسة محفوظة في `.wwebjs_auth/` — ما تحتاج تمسح QR كل مرة.
- لتغيير المنفذ: `PORT=4000 npm start`
- لفصل الرقم: احذف مجلد `.wwebjs_auth/` وأعد التشغيل.
